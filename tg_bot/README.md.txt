# Quiz Bot

Quiz Bot - это простой Telegram-бот, который проводит викторину с вопросами с несколькими вариантами ответов.

## Команды

### /start
Запускает бота и приветствует пользователя. Также предоставляет возможность начать викторину.

### /quiz
Начинает викторину. Эта команда может использоваться для непосредственного начала викторины.

## Функциональность бота

- Бот предлагает пользователю вопросы с несколькими вариантами ответов.
- Пользователь может выбрать ответ из предложенных вариантов.
- Бот отвечает, правильно ли выбранный ответ или нет.
- Бот отслеживает количество правильных ответов пользователя и текущий номер вопроса.
- В конце викторины бот отображает общее количество правильных ответов.
